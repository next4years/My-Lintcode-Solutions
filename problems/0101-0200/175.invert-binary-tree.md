## 题目

### [175. Invert Binary Tree](https://www.lintcode.com/problem/invert-binary-tree/description)

Invert a binary tree.Left and right subtrees exchange.

### Example

**Example 1:**

```
Input: {1,3,#}
Output: {1,#,3}
Explanation:
	  1    1
	 /  =>  \
	3        3
```

**Example 2:**

```
Input: {1,2,3,#,#,4}
Output: {1,3,2,#,4}
Explanation: 
	
      1         1
     / \       / \
    2   3  => 3   2
       /       \
      4         4
```

### Challenge

Do it in recursion is acceptable, can you do it without recursion?

## 思路

两种实现

一种递归；一种非递归

## 代码

```python
"""
Definition of TreeNode:
class TreeNode:
    def __init__(self, val):
        self.val = val
        self.left, self.right = None, None
"""

class Solution:
    """
    @param root: a TreeNode, the root of the binary tree
    @return: nothing
    """
    def invertBinaryTree(self, root):
        # write your code here
        return self.Loop(root)
    
    def recurrence(self, root):
        if not root:
            return None
        
        left = self.recurrence(root.left)
        right = self.recurrence(root.right)
        
        root.left = right
        root.right = left
        
        return root
    
    def Loop(self, root):
        if not root:
            return None
        
        queue = [root]
        
        while queue:
            node = queue.pop()
            left = node.left
            right = node.right
            if left:
                queue.append(left)
            if right:
                queue.append(right)
            node.left = right
            node.right = left
        return root
```
