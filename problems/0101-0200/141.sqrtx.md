## 题目

### [141. Sqrt(x)](https://www.lintcode.com/problem/sqrtx/description)

Implement int `sqrt(int x)`.

Compute and return the square root of *x*.

### Example

```
Example 1:
	Input:  0
	Output: 0


Example 2:
	Input:  3
	Output: 1
	
	Explanation:
	return the largest integer y that y*y <= x. 
	
Example 3:
	Input:  4
	Output: 2
	
```

### Challenge

O(log(x))

## 思路

找到所有满足平方小于等于x的数的最后一个位置的数

## 代码

```python
class Solution:
    """
    @param x: An integer
    @return: The sqrt of x
    """
    def sqrt(self, x):
        # write your code here
        start, end = 0, x
        while start + 1 < end:
            mid = (start + end) // 2
            if mid**2 > x:
                end = mid
            elif mid**2 == x:
                return mid
            else:
                start = mid
        if end**2 <= x:
            return end
        if start**2 <= x:
            return start
```
