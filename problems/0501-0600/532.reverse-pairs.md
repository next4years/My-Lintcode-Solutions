## 题目

### [532. Reverse Pairs](https://www.lintcode.com/problem/reverse-pairs/description)

Reverse pair is a pair of numbers (A[i], A[j]) such that A[i] > A[j] and i < j. Given an array, return the number of reverse pairs in the array.

### Example

**Example1**

```
Input:  A = [2, 4, 1, 3, 5]
Output: 3
Explanation:
(2, 1), (4, 1), (4, 3) are reverse pairs
```

**Example2**

```
Input:  A = [1, 2, 3, 4]
Output: 0
Explanation:
No reverse pair
```

## 思路

利用归并排序的思想求逆序对，复杂度O(nlogn)

也可以用树状数组或者线段树求解

一般做法是在归并排序的过程中进行逆序对的计算，这里我的代码是先排序，再计算，时间上不是最优的



## 代码

```python
class Solution:
    """
    @param A: an array
    @return: total of reverse pairs
    """
    def reversePairs(self, A):
        # write your code here
        res = self.helper(A)
        return res
    
            
    def helper(self, A):
        
        l = len(A)
        res = 0
        if l == 0 or l == 1:
            return res
        
        subA, subB = A[0: l//2], A[l//2: l]
        m = self.helper(subA)
        n = self.helper(subB)
        #print(subA, subB, m, n)
        res = m + n
        ind1, ind2 = 0, 0
        subA.sort()
        subB.sort()
        while ind1 < len(subA):
            if subA[ind1] > subB[ind2]:
                ind2 += 1
                if ind2 == len(subB):
                    res += len(subB)*(len(subA)-ind1)
                    break                
            else:
                res += ind2
                ind1 += 1

        return res
```

* 归并排序的过程中进行逆序对的计算

  归并排序是将数列a[l,h]分成两半a[l,mid]和a[mid+1,h]分别进行归并排序，然后再将这两半合并起来。
  在合并的过程中（设l<=i<=mid，mid+1<=j<=h），当a[i]<=a[j]时，并不产生逆序数；当a[i]>a[j]时，在
  前半部分中比a[i]大的数都比a[j]大，将a[j]放在a[i]前面的话，逆序数要加上mid+1-i。因此，可以在归并
  排序中的合并过程中计算逆序数.

```python
class Solution:
    """
    @param A: an array
    @return: total of reverse pairs
    """
    def reversePairs(self, A):
        # write your code here
        tmp = [0] * len(A)
        self.count = 0
        self.mergesort(A, 0, len(A) - 1, tmp)
        return self.count
    
    def mergesort(self, A, start, end, tmp):
        if start >= end:
            return
        
        mid = start + (end - start) // 2
        self.mergesort(A, start, mid, tmp)
        self.mergesort(A, mid + 1, end, tmp)
        self.merge(A, start, end, tmp)
        
    def merge(self, A, start, end, tmp):
        mid = start + (end - start) // 2
        i = start
        j = mid + 1 
        index = start
        while i <= mid and j <= end:
            if A[i] > A[j]:
                tmp[index] = A[j]
                j += 1 
                self.count += mid - i + 1 
            else:
                tmp[index] = A[i]
                i += 1 
                
            index += 1 
        
        while i <= mid:
            tmp[index] = A[i]
            i += 1 
            index += 1 
            
        while j <= end:
            tmp[index] = A[j]
            j += 1 
            index += 1 
        
        for i in range(start, end + 1):
            A[i] = tmp[i]
```
